### Instructions:
>**Launching a ChatBox in the Tor network** <br>
First, install Termux from the playmarket. After in Termux, we writing commands: <br>
- `pkg up`
- `pkg install git`
- `git clone https://github.com/7ife/cbdw`
- `cd cbdw`
- `bash s.sh` <br>
